# VPS Credentials After Purchase

## Where to get Server IP and Password

After purchasing a VPS, the hosting provider automatically creates the server and provides:

- Server IP
- Username (usually root)
- Password (auto-generated)
- Operating System details

## Hostinger Path

hPanel → VPS → Manage → Overview

## Connect

ssh root@SERVER_IP

## Change Password

passwd

## Store credentials securely

Use Bitwarden or another password manager. Never store passwords in public repositories.
