# Linux Basics

## What is Linux?

Linux is an open-source operating system widely used in servers, cloud platforms, and enterprise environments.

## Why Linux?

- Open Source
- Secure
- Stable
- High Performance
- Preferred for Production Servers

## Common Linux Commands

| Command | Description |
|---------|-------------|
| pwd | Present Working Directory |
| ls | List files |
| cd | Change directory |
| mkdir | Create directory |
| touch | Create file |
| cp | Copy files |
| mv | Move or rename |
| rm | Remove file |
| cat | View file |
| clear | Clear terminal |
| history | Show command history |
